Web-Based To-Do List

Overview

A full-stack To Do List web application utilizing Django Framework.
This project implements core functionalities of a simple To Do application aiming to improve understanding in full-stack web development and database integration.

Key Features Implemented

Full CRUD Functionality

Implemented comprehensive Create, Read, Update, and Delete operations for tasks, allowing users to fully manage their to-do items from creation to deletion.

Task Status Management

Enables users to easily categorize tasks as To Do or Completed, providing clear visual separation and organization within the task list. Additionally, provides controls to easily mark tasks as complete or incomplete directly from the main list, enhancing user interaction and workflow efficiency.

Detailed Task Attributes

Displays essential task details including due dates, completion dates, and priority levels for effective task prioritization and tracking.

Overdue Task Highlighting

Automatically identifies and highlights overdue tasks at the top of the list, ensuring critical items are immediately brought to the user's attention.

Task Search & Filtering

Includes a simple search bar to quickly find specific tasks by keyword, significantly improving navigation and efficiency for larger task lists.
